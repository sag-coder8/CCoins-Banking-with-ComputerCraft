# CCoins-Banking-with-ComputerCraft
A digital banking system for a Minecraft SMP with ComputerCraft, should be kept semi-updated
Intended use: Put bank_client on advanced pocket computers and label with names to be used in transactions, then put startp.lua in the startup folder of the root directory for automatic updates on restart and put update.lua in the root directory, then run bank_server.lua on a server advanced computer
